# Pixel Art Studio

A comprehensive pixel art editor designed for game developers and digital artists. Create, edit, and export pixel art with precision and ease.

## Features

- **Grid System**: Adjustable grid size for precise pixel placement
- **Color Palette Management**: Built-in palette with color picker and palette editor
- **Multiple Drawing Tools**: Pencil, eraser, fill bucket, and selection tools
- **Layer Support**: Work with multiple layers for complex artwork
- **Zoom Controls**: Zoom in and out for detailed editing
- **Export Options**: Save as PNG, GIF, or SVG formats
- **Canvas Resizing**: Adjust canvas size for different projects
- **Color Swapping**: Swap colors in your artwork
- **Undo/Redo**: Full editing history support
- **Real-time Preview**: See changes instantly as you work

## How It Works

1. **Set Canvas Size**: Define your canvas dimensions in pixels
2. **Choose Tools**: Select from the available drawing tools
3. **Select Colors**: Pick colors from the palette or use the color picker
4. **Draw**: Create your pixel art using the drawing tools
5. **Edit**: Use layers, selection tools, and color swapping for advanced editing
6. **Export**: Save your artwork in your preferred format

## Usage Instructions

1. **Canvas Setup**: 
   - Set canvas width and height in pixels
   - Adjust grid size for precision
2. **Drawing Tools**:
   - **Pencil**: Draw individual pixels
   - **Eraser**: Remove pixels
   - **Fill Bucket**: Fill areas with selected color
   - **Selection**: Select and manipulate parts of your artwork
3. **Color Management**:
   - Choose colors from the palette
   - Use the color picker for custom colors
   - Add colors to your palette
4. **Layers**:
   - Create and manage multiple layers
   - Adjust layer opacity
   - Merge layers when needed
5. **Export**:
   - Save as PNG for high-quality images
   - Save as GIF for animated pixel art
   - Save as SVG for vector versions

## Controls

- **Canvas Size Inputs**: Set width and height in pixels
- **Grid Size Selector**: Adjust grid precision
- **Tool Selection**: Choose drawing tools
- **Color Palette**: Select and manage colors
- **Layer Controls**: Manage multiple layers
- **Zoom Controls**: Zoom in/out for detailed editing
- **Export Buttons**: Save artwork in various formats
