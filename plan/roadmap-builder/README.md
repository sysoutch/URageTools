# Roadmap Builder

![Thumbnail](thumbnail.png)

This tool helps you create and visualize project roadmaps. It's useful for planning software development, game projects, or any multi-phase initiative.

## Features

- Create visual roadmaps with milestones and tasks
- Customize timeline and phases
- Add dependencies between tasks
- Export roadmaps as images
- Responsive design for various screen sizes

## How to Use

1. Create a new roadmap
2. Add phases and milestones
3. Add tasks with start/end dates
4. Set dependencies between tasks
5. Visualize your roadmap
6. Export as image when complete

## Technical Details

- Client-side web application
- Visual timeline representation
- Responsive design that works on mobile and desktop
- No server-side processing required
- Export functionality for sharing roadmaps
