# QR Code Creator

This tool generates QR codes from text, URLs, or other data. It's useful for creating shareable links, contact information, or any data that can be encoded in a QR code.

## Features

- Generate QR codes from text or URLs
- Customize QR code size and colors
- Download QR codes as images
- Preview before downloading
- Responsive design for various screen sizes

## How to Use

1. Enter the text or URL you want to encode
2. Adjust size and color settings if desired
3. Preview the QR code
4. Download the QR code as an image
5. Share the QR code with others

## Technical Details

- Uses QR code generation library
- Supports various QR code formats
- Downloadable as PNG images
- Responsive design that works on mobile and desktop
- No server-side processing required
