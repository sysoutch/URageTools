# Meta OG Tag Generator

This tool generates Open Graph (OG) tags for web pages to improve social media sharing. It helps create rich previews when links are shared on platforms like Facebook, Twitter, and LinkedIn.

## Features

- Generate OG tags for various content types
- Customize title, description, image, and other metadata
- Preview of how the tag will appear on social media
- Export generated tags as HTML code
- Responsive design for various screen sizes

## How to Use

1. Enter your page title and description
2. Upload or enter the image URL for sharing
3. Customize additional metadata fields
4. Preview how the tag will appear
5. Copy the generated HTML code to your webpage

## Technical Details

- Generates proper OG meta tags for social media platforms
- Responsive design that works on mobile and desktop
- No server-side processing required
- Client-side JavaScript implementation
