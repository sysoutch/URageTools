# Dialogue Tree Visualizer

This tool helps visualize dialogue trees for games, interactive stories, or any application that requires branching narrative structures. It allows you to create, edit, and visualize complex dialogue systems.

## Features

- Create and edit dialogue nodes with text and choices
- Visualize dialogue trees with connected nodes
- Export dialogue trees as images or JSON
- Responsive design for various screen sizes
- Drag and drop functionality for node positioning

## How to Use

1. Click "Add Node" to create a new dialogue node
2. Enter text for the node
3. Connect nodes using the connection tool
4. Add choices to nodes to create branches
5. Export your dialogue tree when complete

## Technical Details

- Built with HTML5 Canvas for rendering
- Uses JavaScript for interactive functionality
- Responsive design that works on mobile and desktop
- No external dependencies required
