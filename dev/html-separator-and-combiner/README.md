# HTML Separator and Combiner

![Thumbnail](thumbnail.png)

A tool that allows you to separate HTML elements into individual files and combine them back into a single HTML document.

## Features

- **Separate HTML**: Extract individual HTML elements from a single document
- **Combine HTML**: Merge multiple HTML files into one complete document
- **Preserve Structure**: Maintain the original HTML structure during separation and combination
- **Easy to Use**: Simple interface for managing HTML components
- **Responsive Design**: Works on various screen sizes

## How to Use

1. Select the HTML file you want to separate or combine
2. For separation: Choose the elements you want to extract
3. For combination: Select the HTML files to merge
4. Download the resulting HTML file
