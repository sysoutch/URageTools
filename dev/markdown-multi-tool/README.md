# Markdown Multi-Tool

This comprehensive tool provides multiple markdown-related functionalities in one interface. It helps with formatting, conversion, and manipulation of markdown content.

## Features

- Markdown to HTML conversion
- HTML to Markdown conversion
- Real-time preview of markdown content
- Syntax highlighting
- Table generator
- Code block formatting
- Responsive design for various screen sizes

## How to Use

1. Enter or paste your markdown content in the input area
2. See the real-time HTML preview
3. Use the conversion buttons to switch between formats
4. Generate tables or code blocks using the provided tools
5. Copy the final result to your clipboard

## Technical Details

- Client-side processing (no server required)
- Real-time preview using JavaScript
- Supports all common markdown syntax
- Responsive design that works on mobile and desktop
- No external dependencies required
