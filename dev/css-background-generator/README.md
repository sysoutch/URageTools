# CSS Background Generator

![Thumbnail](thumbnail.png)

A tool for creating beautiful and customizable CSS background styles with gradients, patterns, and effects.

## Features

- **Gradient Generator**: Create linear and radial gradients with multiple color stops
- **Pattern Library**: Choose from various pre-designed background patterns
- **Texture Effects**: Add textures like noise, canvas, or fabric to backgrounds
- **Customizable Properties**: Adjust opacity, size, position, and repeat settings
- **Real-time Preview**: See changes instantly as you adjust parameters
- **CSS Code Export**: Generate clean, copy-paste ready CSS code
- **Responsive Design**: Backgrounds adapt to different screen sizes

## How to Use

1. Select a background type (gradient, pattern, or texture)
2. Customize the properties using the control panel
3. Preview your background in real-time
4. Copy the generated CSS code to use in your projects
