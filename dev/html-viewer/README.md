# HTML Viewer

![Thumbnail](thumbnail.png)

A simple tool to view and preview HTML files directly in your browser with live reloading capabilities.

## Features

- **Live Preview**: See changes to your HTML files in real-time
- **Responsive Design**: Works on all screen sizes
- **Easy to Use**: No complex setup required
- **File Loading**: Load and display HTML files from your local system

## How to Use

1. Open `index.html` in your web browser
2. The tool will automatically load and display the HTML content
3. Any changes you make to the HTML file will be reflected immediately in the preview

## Files

- `index.html` - Main HTML file with the viewer interface
- `style.css` - CSS styling for the viewer
- `script.js` - JavaScript for handling file loading and preview

## Requirements

- Modern web browser (Chrome, Firefox, Edge, Safari)
- No additional dependencies required

## Development

To modify the HTML viewer:

1. Edit `index.html` to change the interface
2. Modify `style.css` to update the styling
3. Update `script.js` to add new functionality
